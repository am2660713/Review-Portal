# Performance Review Workflow

## Flow implemented
1. Admin logs in and allocates review assignments (employee + manager + year).
2. Employee logs in and sees only their assigned review forms.
3. Employee submits Self Review (aligned with your Self-Review Excel sections).
4. Manager logs in and sees assigned employees.
5. Manager submits Manager's Review (quality, productivity, job knowledge, comments).
6. Admin sees all managers and assignment/review status.

## Run
- `npm run dev`
- Frontend: `http://localhost:5174/`
- Backend API: `http://localhost:3000/`
